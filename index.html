    Instruções Importantes do Thony

    Eu (Thony) provavelmente estou dormindo, então sigam essas regras até eu acordar:

    1. Não façam login os dois ao mesmo tempo para evitar conflitos de dados.
    2. Se quiserem, podem apagar os contratos de exemplo que estão aí.
    3. A senha para Excluir ou Resetar Pagamentos é a mesma senha do login.
    4. Preencham o ID do Cliente na ordem correta, de acordo com a planilha de vocês.
    5. Coloquem a Data do Cadastro original de cada cliente, mesmo para os que já estão em renovação (depois explico como funciona para renovar).
    6. O botão Lembrete de Cobrança só funciona para clientes com status "Pendente". Ele gera a mensagem e o link do WhatsApp.
    7. Para o Lembrete funcionar, o cliente precisa estar com o número de celular correto no cadastro e salvo no celular que vai enviar a mensagem.
    8. Sugiro começarem adicionando os clientes mais recentes, ou aqueles que têm parcelas pendentes ou começaram a pagar há pouco tempo. Assim já podem ir atualizando o calendário de pagamentos deles.
    9. Podem preencher os dados dos clientes sem medo. Depois eu ajudo a arrumar qualquer detalhe do calendário se for preciso.

    Qualquer dúvida, me chame quando eu acordar. Bom trabalho!
